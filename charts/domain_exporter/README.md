
Domain_exporter
===========

Exports the expiration time of your domains as prometheus metrics.

This Helm chart deploys the [caarlos0/domain_exporter](https://github.com/caarlos0/domain_exporter).

## Prerequisites

- Kubernetes 1.19+
- Helm 3

## Installing the Chart

To install the chart with the release name `my-domain_exporter`:

```bash
helm repo add evilgn0me https://evilgn0me.github.io/helm-charts/
helm install my-domain_exporter evilgn0me/domain_exporter
```
